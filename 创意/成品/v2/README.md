# 创意玩法知识网络 v2

当前最新小版本：`v2.7`。

v2.x 从 `创意/资料源/*_classified*.json` 全量重建，不从 v1 世界书追加继承。

主要产物：

- `极品调教大世界书_v2.json`：导入 SillyTavern 的主世界书。
- `极品调教快捷回复_v2_来源无状态版.json`：按来源类型、小说名浏览玩法。
- `极品调教快捷回复_v2_十二卷无状态版.json`：按玩法分类浏览玩法；文件名沿用十二卷版以保持导入路径稳定。
- `资料源索引_v2.json`：来源、分类、跳过原因和 UID 对照。
- `重建报告.md`：给人看的重建摘要。

公开仓库只保存发布产物和脱敏索引；本地资料源路径不写入发布版索引。
